## 追加実装機能

以下は、開発要件に記載のない追加実装機能です。

### 会員登録
- メール認証誘導画面で認証メール再送後、「認証メールを再送信しました」を表示
- 登録済みメールアドレスで会員登録しようとした場合、「このメールアドレスは既に登録されています」と表示

### 勤怠打刻
- 同日に2度目の出勤打刻を使用とした場合、「本日は既に出勤済みです」と表示

### 勤怠修正
- 管理者が勤怠修正後、「勤怠データを更新しました」を表示
- 休憩開始時刻が終了時刻より後の場合、「休憩時間が不適切な値です」を表示
― 休憩開始時刻と休憩終了時刻が退勤時刻より後の場合、「休憩時間が不適切な値です」（休憩開始に対し）と「休憩時間もしくは退勤時間が不適切な値です」（休憩終了に対し）が二重で表示されますが、要件に従いそのままの仕様としています。
